🔧 HOW THE CODE WORKS:
────────────────────────────────────────────────────────────────────────────────

pricing.html (JavaScript at bottom of page):

```javascript
document.addEventListener('DOMContentLoaded', function() {
    const loginRequired = document.getElementById('loginRequired');
    const pricingContent = document.getElementById('pricingContent');
    
    if (auth.isLoggedIn()) {
        // User is logged in - show pricing
        loginRequired.style.display = 'none';
        pricingContent.style.display = 'block';
    } else {
        // User is not logged in - show login prompt
        loginRequired.style.display = 'block';
        pricingContent.style.display = 'none';
    }
});
```

This code:
1. Runs when page loads
2. Checks if user is logged in (auth.isLoggedIn())
3. If YES: Shows full pricing tables
4. If NO: Shows "Member Access Required" message with login/signup buttons
