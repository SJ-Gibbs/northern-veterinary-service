## 🐛 Troubleshooting:

### **If "Run" button doesn't work:**
1. Check the Console/Shell tab for errors
2. Manually run: `node server.js`
3. Check that port 3000 is not already in use

### **If website doesn't load:**
1. Verify server is running (check Console)
2. Look for the message: "Server running on port 3000"
3. Click the Webview panel or open in new tab
4. Use the Replit public URL (not localhost)

### **If you see "command not found":**
- Replit needs to install Node.js
- Wait for the environment to build
- Or manually install: `nix-env -iA nixpkgs.nodejs`
