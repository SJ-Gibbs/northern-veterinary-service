💡 COMMON ISSUES & SOLUTIONS:
────────────────────────────────────────────────────────────────────────────────

Issue: "I see old content"
→ Solution: Hard refresh (Ctrl+Shift+R) or clear browser cache

Issue: "Login/Signup buttons don't appear"
→ Check: Browser console (F12) for JavaScript errors
→ Check: auth.js is loaded properly

Issue: "Prices show even when not logged in"
→ Check: Browser localStorage might have a session
→ Solution: Clear localStorage or try incognito/private window

Issue: "Changes I see in files don't appear on preview"
→ Check: Make sure files are saved
→ Solution: Hard refresh the preview
→ Check: Server is serving from deploy/ folder (it is!)
