┌──────────────────────────────────────────────────────────────────────────────┐
│  🔐 HOW AUTHENTICATION WORKS                                                 │
└──────────────────────────────────────────────────────────────────────────────┘

1. SIGNUP:
   User fills form → Validation → Account created → Auto-login → Redirect

2. LOGIN:
   User enters credentials → Verification → Session created → Redirect

3. PROTECTED CONTENT:
   User visits pricing → Check if logged in → Show/hide content

4. LOGOUT:
   User clicks logout → Clear session → Redirect to homepage

5. SESSION:
   Login status stored in browser localStorage → Persists across pages
